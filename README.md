## Hardware:
1. 3 x HS-5485HB Servo (https://www.servocity.com/hs-5485hb-servo)
2. Frame to hold the camera
3. Bottom frame to hold the servos
4. 1 x PS3 Eye
5. 2 x PWM Servo Driver (https://www.adafruit.com/product/815)
